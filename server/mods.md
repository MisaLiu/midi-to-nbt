## Server mod list

List of mods used in this project.

* [KubeJS](https://modrinth.com/mod/kubejs) - Project base
* [C2ME](https://modrinth.com/mod/c2me-fabric) - Create/load world much faster
* [Starlight](https://modrinth.com/mod/starlight) - Avoid light update spike
* [Alternate Current](https://modrinth.com/mod/alternate-current) - Avoid redstone related spike
* [Carpet mod](https://modrinth.com/mod/carpet) - Adjust game tickrate
* [G4mespeed](https://modrinth.com/mod/g4mespeed) - Sync server-client tickrate
    * Turn off the `Shift sound pitch` setting!
* [Litematica](https://modrinth.com/mod/litematica) - Easiest way to place structures
